Project Description

贸易宝 Digital Trader 为中国出口贸易服务，贸易从业者期望能有与之配套的跨境贸易支付解决方案，为他们解决境外付款、合规外汇结汇的实际问题。

Competitive Analysis

调研提供类似服务的产品为设计如何开始提供了一个线索。我们从产品服务、客户群、使用流程三个维度出发，对 AirWallex、Wise、Payoneer进行调研分析。

Designing user interaction flow

在第一个版本中，用户发起付款请求的流程是单页线性操作的，这很简单直接。但很快我们遇到发现，贸易汇款一般涉及金额较大，用户在付款请求界面操作时，由于填写的内容都在一页，且主次区分不明显，导致他们不断的来回小心核对信息，增加了体验负担。于是我在尝试把流程拆分，让用户在三个步骤聚焦关键信息——金额，付款细节，分享。

Additional Design

据我们用户分析，国际客户很大一部分还是比较依赖邮件沟通，因此我们也对邮件通知模版重新设计以传达给用户品牌信任感。

Conclusion

国际跨境支付是一项颇具挑战性且有意思的业务，涉及国家外汇管理，银行流程操作，客户各种不同商业需求等。尽管在产品界面层面，呈现给用户的是简洁明了的交互流程，但其背后的产品逻辑，数据接口，银行跨平台间的数据交换，汇率波动因素等等，都是需要在设计过程中不断的理清和考虑的。

这个项目作为 Digital Trader 核心盈利部分，已完成了数千万乃至上亿美金的资金结汇。
